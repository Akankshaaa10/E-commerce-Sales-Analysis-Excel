# E-commerce Sales Analysis (Excel)

## Project Overview
This project analyzes e-commerce sales data using Microsoft Excel to understand overall business performance.  
The goal of the analysis is to identify key sales trends, evaluate product category performance, and highlight insights that can support data-driven business decisions.

The project demonstrates how Excel can be used to perform exploratory data analysis and build interactive dashboards using pivot tables and charts.

---

## Business Questions

The analysis aims to answer the following questions:

- What is the overall sales performance of the business?
- Which product categories generate the highest revenue?
- How do sales change over time?
- Which products contribute the most to total sales?
- What trends can be observed in customer purchasing behavior?

---

## Tools Used

**Microsoft Excel**
- Data cleaning

- Dashboard creation
- Data visualization

---

## Data Analysis Process

The project followed these main steps:

1. Data exploration and understanding
2. Data cleaning and preparation
3. Creation of pivot tables for analysis
4. Development of visual charts to identify patterns
5. Building an interactive Excel dashboard

---

## Dashboard Features

The Excel dashboard includes:

- Total Sales KPI
- Product Category Sales Analysis
- Sales Trend Over Time
- Top Performing Products
- Interactive filters for better exploration

---

## Key Insights

- Certain product categories contribute significantly more revenue than others.
- Sales show seasonal trends across different months.
- A small number of products generate a large share of total sales.
- Visual analysis helps identify opportunities to optimize product performance.
