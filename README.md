# E-commerce Sales Analysis (Excel)

## Project Overview
This project analyzes an e-commerce sales dataset using Microsoft Excel to uncover insights about sales performance across regions, product categories, and individual products. The goal of this project is to transform raw sales data into meaningful business insights through data cleaning, pivot table analysis, and dashboard visualization.

## Dataset
The dataset contains 1200 sales records with the following fields:

- Order_ID 
- Order_Date 
- Customer_ID 
- Region 
- Category 
- Product 
- Quantity 
- Price 
- Revenue 


## Analysis Performed
The dataset was analyzed to answer key business questions:

- Which regions generate the highest revenue
- Which product categories contribute the most sales
- Which products are top performers
- How revenue changes across months

Pivot tables were used to summarize and analyze the data across these dimensions.

## Dashboard
An interactive Excel dashboard was created to visualize key metrics including:

- Total Revenue
- Total Orders
- Revenue by Region
- Revenue by Category
- Top Products
- Monthly Revenue Trends

The dashboard provides a quick overview of business performance and helps identify patterns in sales data.

## Tools Used
- Microsoft Excel
